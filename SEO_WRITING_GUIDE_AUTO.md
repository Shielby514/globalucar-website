# Globalucar / Kingmay SEO 写作规范（汽车出口·汽配行业版）
> 基于领动SEO核心逻辑，结合汽车出口/汽配行业特性定制
> 生效日期：2026-04-28，适用于所有新增及更新文章

---

## 一、核心目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 关键词密度 | **1.5% ~ 3%** | 汽车行业标准，比家具行业更克制，避免SEO痕迹过重 |
| 关键词种类 | **≥ 3 种变体** | 核心词 + 变体词（同义词/车型全名与简称/参数词） |
| 内链数量 | **≥ 3 条/篇** | 文章 → 产品详情页 + 同品类姊妹文章 |
| CTA结尾 | **必须** | 每篇文章末尾含询盘引导 + 双品牌露出 |
| 文章字数 | **≥ 800 英文词** | 深度内容优先，1600词为优秀标准 |

## 二、关键词策略（汽车/汽配行业特化）

### 2.1 关键词配置公式

```
核心词 + 1~2个修饰词
```

**汽车整车类文章示例：**
| 核心词 | 变体词示例 |
|--------|-------------|
| Geely Coolray | Geely Coolray price / Coolray SUV export / Geely Coolray FOB China / buy Coolray from China |
| BYD ATTO 3 | BYD ATTO 3 price / BYD electric SUV / ATTO 3 export to Middle East |
| used cars | import used cars / Chinese used cars / second hand cars China / used cars wholesale |

**汽配类文章示例：**
| 核心词 | 变体词示例 |
|--------|-------------|
| auto parts | OEM auto parts / aftermarket auto parts / car parts wholesale / vehicle spare parts |
| engine sensors | oxygen sensor / MAF sensor / engine sensor failure / automotive sensors |
| alternator | car alternator / alternator wholesale / alternator OEM / automotive alternator |

### 2.2 关键词种类要求

- 每篇文章**至少配置 5 个关键词变体**（含核心词）
- 变体词须在**标题、H2/H3小标题、正文前30词、结尾段**中自然出现
- 禁止堆砌：单段出现同一变体词不超过 **2 次**

### 2.3 密度控制技巧

| 位置 | 密度策略 |
|------|---------|
| **标题（H1）** | 必须含核心词或核心变体 |
| **导语段（前50词）** | 核心词 + 1个变体词，自然嵌入 |
| **H2/H3小标题** | 至少 50% 的小标题含关键词变体 |
| **正文段落** | 每段 1 次关键词变体，均匀分布 |
| **结尾段** | 核心词 + 2个变体词 + CTA引导 |

## 三、文章结构标准

### 3.1 标准结构（金字塔原理：结论先行）

```
开头（Hook + 核心结论）
  ↓
文章纲要（3~5个要点，用 <ul> 或 <p> 列出）
  ↓
正文（每节 H2 小标题 + 2~4段正文 + 可选表格/列表）
  ↓
结尾（总结 + CTA + 品牌露出 + 内链）
```

### 3.2 H2/H3 小标题要求

- 每篇文章 **H2 ≥ 3 个**，H3 按需使用
- **50%以上的H2/H3须含关键词变体**
- 小标题写法：适当扩句，嵌入长尾关键词
  - ❌ 差例：`About Price`
  - ✅ 好例：`Geely Coolray Wholesale Price and MOQ for 2026`

### 3.3 正文段落要求

- 每段 **3~5 句**，主题单一
- 每 **2~3 段**使用一次项目符号（`<ul>/<ol>`）或数据表格（`<table>`）
- 段落之间用 **过渡句** 连接，避免生硬跳转
- 英文写作：避免 AI 高频词（streamlined / seamless / unlock / delve）

### 3.4 结尾 CTA 标准模板

```html
<h2>Ready to [核心行动]?</h2>
<p>[2~3句：总结文章核心价值 + 引导询盘]</p>
<p>
  <a href="contact.html">Contact our team</a> for a tailored quote.
  <a href="products.html">Browse all [产品类别]</a> or
  <a href="https://wa.me/86133xxxx">WhatsApp direct</a>.
</p>
<p><strong>Globalucar / Kingmay Auto Parts</strong> — Your trusted partner in [市场/品类].</p>
```

## 四、内链策略（内容群策略）

### 4.1 内链分布要求

| 位置 | 内链目标 | 数量 |
|------|---------|------|
| 开头段 | 品牌首页 `index.html` 或关于页 `about.html` | 0~1 |
| 正文段落 | **对应产品详情页** `product-detail.html?id=XXX` | 2~3 |
| 结尾段 | 产品列表页 `products.html?cat=xxx` + 联系页 | 1~2 |
| 侧边/文末 | 同品类姊妹文章（blog-post.html?slug=xxx）| 1~2 |

**最低要求：每篇文章内链总数 ≥ 3 条**

### 4.2 内容群互链示例（Coolray 促销文章）

```
id=15 正文：
  → product-detail.html?id=121  (Coolray Basic)
  → product-detail.html?id=203  (Coolray Battle)
  → product-detail.html?id=204  (Coolray Full Option)
  → products.html?cat=new-cars
  → blog-post.html?slug=how-to-import-new-chinese-cars-2026  (姊妹文章)
```

## 五、品牌词处理规则

| 场景 | 处理方式 |
|------|---------|
| **产品标题（H1）** | 减少品牌词，突出车型/零部件名+核心参数 |
| **H2/H3小标题** | 品牌词可出现，但不做主关键词 |
| **正文第一段** | 自然嵌入品牌：Globalucar supplies... / Kingmay provides... |
| **结尾段** | 必须双品牌露出：Globalucar + Kingmay Auto Parts |
| **meta description** | 品牌词可出现 1 次 |

## 六、汽配行业专项规范

### 6.1 零部件关键词写法

- **优先使用买家搜索词**：不是 "OEM 12345 sensor"，而是 "oxygen sensor for Toyota Camry"
- **标注适配车型**：在正文或表格中列出适用车型/年份
- **OEM编号作为补充**：可在表格中列出 OEM 对照号，不作为主关键词

### 6.2 技术参数写作规范

- 尺寸/重量/材质：必须标注单位（mm / kg / ABS/PA66 等）
- 认证标准：IATF 16949 / ISO 9001 可作为信任信号写入
- 测试数据：盐雾测试小时数、疲劳测试周期数——有则写，无则不编

---

## 七、整车出口专项规范

### 7.1 车型关键词写法

- **车型全名 + 年款**：Geely Coolray 2026 / BYD ATTO 3 2025
- **简写变体**：Coolray / ATTO 3（在正文中可与全名交替使用）
- **出口目的地词**：export to [Country] / wholesale [Country] / import [Country]

### 7.2 价格/贸易术语规范

- **FOB / CIF / EXW**：在价格相关内容中必须标注贸易术语
- **MOQ**：每款车型/零部件须标注最小起订量
- **价格有效期**：标注至具体日期（如 valid until 2026-06-30）

## 八、文章更新检查清单

更新/新建文章时，逐项核对：

- [ ] 关键词密度 1.5%~3%（用 density_check.py 检测）
- [ ] 关键词变体 ≥ 3 种（实际出现在正文中的）
- [ ] H2/H3 小标题 ≥ 50% 含关键词变体
- [ ] 内链 ≥ 3 条（含至少 1 条产品详情页链接）
- [ ] 结尾有 CTA + 双品牌露出
- [ ] 文章字数 ≥ 800 词
- [ ] 无 AI 高频词（streamlined/seamless/unlock/delve 等）
- [ ] 图片 alt 属性含关键词变体
- [ ] meta description 含核心词，长度 150~160 字符
