## 一、竞品分析

### 1.1 竞品网站 VenavaParts（venavaparts.com）分析

| 维度 | 分析 |
|------|------|
| 产品规模 | **786个SKU**，66页产品列表 |
| 品牌覆盖 | 16个品牌（现代/起亚为主力，丰田/日产/奥迪/BMW/Benz等） |
| 产品分类 | 品牌→品类双维分类（空气悬架/传动轴/轴承/连杆/曲轴/气弹簧/大灯总成/油泵/火花塞等） |
| 价格区间 | $17 ~ $180，折扣力度 25%~44% |
| 命名规范 | 原厂格式（如 "COVER ASSY-TIMING BELT FRT,RH"） |
| 视觉风格 | 白灰简洁工业风，产品图白底 |
| 内容短板 | ⚠️ 无视频内容 / ⚠️ 无博客更新 / ⚠️ 无车型适配指南 / ⚠️ 无社交媒体运营 |

**可借力点：**
- 786个SKU的产品分类体系 → 可直接作为我们内容规划的**产品库**
- 原厂命名格式 → 视频标题可用OEM编号增强专业感
- 适配多车型信息 → 内容中强调"兼容性"解决买家痛点

### 1.2 行业社媒竞品对标

| 竞品类型 | 代表账号 | 内容策略 | 弱点 |
|---------|---------|---------|------|
| 汽配批发商 | 各B2B INS/FB账号 | 产品图+价格，硬广为主 | 内容枯燥，无知识性，互动低 |

## 三、封面统一风格方案

### 3.1 封面设计规范（豆包AI生成）

**视觉元素：**
- 尺寸：1080×1920（竖屏，适配所有平台）
- 背景：深蓝渐变（#0d1f3c → #1a3a6e，与网站品牌色一致）
- 产品图：居中放置，白底抠图，占画面40-50%
- 系统色标（左上角圆形徽章）：

| 系统 | 颜色 | 标签 |
|------|------|------|
| 🔴 Engine | #e74c3c | ENGINE SYSTEM |
| 🔵 Cooling | #3498db | COOLING SYSTEM |
| 🟡 Suspension | #f39c12 | SUSPENSION SYSTEM |
| ⚡ Electrical | #9b59b6 | ELECTRICAL SYSTEM |

**文字规范：**
- 顶部标题栏：白字，英文 28pt 粗体
- 底部信息栏：零件编号（OEM）+ 适配车型，灰字 18pt
- 右下角：Kingmay logo 水印

### 3.2 封面模板示例

```
┌─────────────────────────┐
│  [🔴 ENGINE SYSTEM]      │  ← 系统色标
│                          │
│     HOW TO IDENTIFY      │  ← 标题（英文）
│    A QUALITY WATER       │
│       PUMP               │
│                          │
│      ┌──────────┐        │
│      │          │        │
│      │  产品图   │        │  ← 白底抠图
│      │          │        │
│      └──────────┘        │
│                          │
│  OEM: 25100-27000        │  ← 零件编号
│  Fits: Toyota Camry 2018+│  ← 适配车型
│                    🏷️Kingmay│
└─────────────────────────┘
```

### 3.3 豆包生成流程

1. **产品图准备**：从 venavaparts.com 或 assets/parts/ 获取白底产品图
2. **Prompt 模板**：
   > "Create a product showcase thumbnail for social media. Dark blue gradient background (#0d1f3c to #1a3a6e). [PRODUCT IMAGE] centered with white cutout background. Top: [ENGLISH TITLE] in bold white text. Bottom-left: '[OEM NUMBER] | Fits: [CAR MODELS]' in gray text. Bottom-right: 'Kingmay' watermark. Modern, professional automotive parts aesthetic. 1080x1920px."
3. **批量生成**：每次准备5-10个产品图，一次性生成当周封面

## 六、AI 视频制作工作流

### 6.1 单条视频制作流程（SOP）

```
Step 1: 素材准备（10分钟）
├── 从 venavaparts.com / assets/parts/ 下载产品图
├── 确认OEM编号、适配车型、核心参数
└── 整理到当周内容表格

Step 2: 文案生成 — 豆包 AI（5分钟）
├── 输入：产品名 + OEM编号 + 适配车型 + 核心卖点
├── 输出：30-60秒视频脚本（英文）+ 文案标题 + Hashtags
└── 模板：
    "Write a 45-second TikTok/Reels script about [PART NAME].
     OEM: [NUMBER]. Fits: [MODELS]. 
     Start with a hook about a common problem this part solves.
     Include 3 key specs. End with CTA to visit kingmay-autoparts.com."

Step 3: 封面生成 — 豆包 AI（5分钟）
├── 输入：产品图 + 标题文字 + 系统色标
├── Prompt 见 3.3 节模板
└── 输出：1080×1920 统一风格封面

Step 4: 视频生成 — AI视频工具（10分钟）
├── 方案A：豆包/可灵 AI 文生视频（产品展示动画）
├── 方案B：产品图 + AI配音 + 字幕 + 背景音乐（CapCut模板）
└── 输出：30-60秒竖屏视频

Step 5: 发布（5分钟）
├── FB #2: 上传视频 + 封面 + 英文文案 + Hashtags
├── INS #2: Reel + 封面 + 英文文案 + Hashtags（15-20个）
├── TikTok: 上传视频 + 英文文案 + Hashtags（8-12个）
└── 三平台同步，文案可微调
```

**单条总耗时：约 35 分钟 / 熟练后可压到 20 分钟**
